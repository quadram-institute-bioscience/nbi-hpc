This is a short hands-on introduction to use the
**NBI HPC** to perform bioinformatics analyses.

This companion websites focuses on the topics that might be of general interest, but
it's tailored for the use at the [Norwich Bioscience Institutes](https://www.nbi.ac.uk/).

## Getting help

> This areas are restricted to NBI staff, and will require authentication

The NBI HPC is managed by [NBI Research Computing](), that
maintains:

* [A ticketing system](https://nbip-research-computing.atlassian.net/servicedesk/customer/portal/), where to raise support requests
* [A wiki](https://nbip-research-computing.atlassian.net/wiki/spaces/RCS/overview), with some documentation about the cluster itself

## Prerequisites

- Linux command line (confident navigation of the filesystem and basic commands)
- Prior experience with SSH connection is optional but recommended
